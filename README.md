# Rock Paper Scissors

A React-based implementation of the classic Rock Paper Scissors game, demonstrating proficiency with multiple CSS styling approaches in React applications.

## Features

- Interactive gameplay against a computer opponent
- Real-time game result feedback
- Responsive design with modern styling

## Technologies Used

- **React** — Component-based UI library
- **CSS Modules** — Scoped styling for maintainability
- **Inline Styles** — Dynamic style application
- **JavaScript ES6** — Modern JavaScript syntax

## Styling Approach

This project showcases three distinct CSS styling techniques in React:
- **Inline Styles** — For dynamic, component-level styling
- **Object Variables** — For reusable style objects
- **CSS Modules** — For scoped, maintainable stylesheets

## Installation

git clone https://github.com/gitTwix/rock-paper-scissors.git
cd rock-paper-scissors
npm install
npm start
